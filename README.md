# Word Sprint

英語版「30秒で10種の花を言う」トレーニングアプリ。認知症予防のカテゴリー想起トレーニングにヒントを得た、英単語の瞬発力を鍛えるクイズです。カテゴリー（お題）を見て、制限時間内にそのカテゴリーに属する英単語をできるだけ多く話す/書く → Geminiが採点・フィードバックします。

## 特徴

- スマホ対応のシンプルなSPA（ビルド不要、素のHTML/CSS/JS）
- 設問モード: 指定 / お任せ（学習目的に合わせて出題） / 完全お任せ
- 入力モード: Speaking（録音してGeminiのマルチモーダル音声理解で採点） / Writing（カンマ区切りで単語入力）
- 制限時間つき回答（デフォルト30秒、変更可）＋途中送信
- お題は事前に複数取得してキャッシュしておくことで、リロード時の待ち時間を短縮
- 録音した音声はその場でのみ聞き直し可能（サーバー保存なし、リロードで消去）

## 使い方

1. [Google AI Studio](https://aistudio.google.com/apikey) で無料のGemini APIキーを取得
2. アプリを開き、APIキーを入力（この端末の `localStorage` にのみ保存され、Gemini APIへ直接送信されます。サーバーは介在しません）
3. 設問モード・入力モード・制限時間を選んで「開始」
4. お題が表示されたら、気に入らなければ🔄リロードで別のお題に変更可能。決定したら回答へ
5. 制限時間内にカテゴリーの英単語を言う/書く。途中で切り上げたい場合は「送信する」を押す
6. Geminiによる採点結果とフィードバックを確認。「もう一度」で同じ設定のまま次のお題へ

## ローカルでの動作確認

ビルド不要のため、任意の静的サーバーで配信するだけで動作します。

```bash
npx serve .
# もしくは
python3 -m http.server 8000
```

## デプロイ（GitHub Pages）

リポジトリルート直下に静的ファイルを配置しているため、GitHub の Settings → Pages で対象ブランチ（例: `main`）のルートを公開するだけでデプロイできます。ビルドステップは不要です。

## 技術構成

- フロントエンドのみの静的サイト（ES Modules、ビルドツールなし）
- Gemini API（`gemini-2.5-flash`）をブラウザから直接呼び出し、構造化出力（`responseSchema`）でお題生成・採点結果を取得
- 録音は `MediaRecorder` / `getUserMedia`、音声はBase64化してGeminiのマルチモーダル入力に送信
